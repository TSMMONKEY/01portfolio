## Installation 1. Upload all files to your web server. - Use FTP, SFTP, or your hosting provider's file manager to upload the files. - Ensure you maintain the directory structure during the upload process. 2. Customize the `index.html` file with your personal information. - Open the file in a text editor or HTML editor. - Look for comments indicating where to add your details (e.g., name, bio, skills). 3. Replace placeholder images with your own in the `assets/images/` directory. - Use images of the same dimensions as the placeholders for best results. - Optimize your images for web to ensure fast loading times. 4. Modify `assets/css/style.css` for custom styling. - Be cautious when editing the CSS to avoid breaking the layout. - Test your changes in multiple browsers to ensure compatibility. ## Customization - Update personal information in `index.html` - Edit sections such as About Me, Skills, Portfolio, and Contact. - Ensure all links (social media, portfolio projects) are correctly updated. - Modify colors and styles in `assets/css/style.css` - Look for CSS variables at the top of the file for easy color scheme changes. - Use browser developer tools to identify classes you want to modify. - Add custom JavaScript in `assets/js/main.js` - Implement additional interactivity or functionality as needed. - Be mindful of potential conflicts with existing scripts. - Replace logo and portfolio images - Use high-quality, professional images that represent your brand and work. - Maintain consistent aspect ratios for a polished look. ## Credits - Bootstrap 5.3.3: https://getbootstrap.com/ - Used for responsive grid system and pre-styled components. - FontAwesome: https://fontawesome.com/ - Provides scalable vector icons that can be customized with CSS. - Google Maps: https://www.google.com/maps - Integrated for displaying location information in the contact section. ## Support For support, please contact [thabanimat24@gmail.com](mailto:thabanimat24@gmail.com) - Include detailed information about your issue in your email. - Specify which browser and device you're using when reporting problems. ## Changelog v1.0 - Initial release - Basic responsive design implemented - Core sections added: About, Skills, Portfolio, Contact - Integration with Bootstrap, FontAwesome, and Google Maps
